    Función Derivada

    \(f(x) = a^x\) \(f'(x) = a^x \ln(a)\)
    \(f(x) = \ln(x)\) \(f'(x) = \frac{1}{x}\)
    \(f(x) = \log_a(x)\) \(f'(x) = \frac{1}{x \ln(a)}\)

    Reglas de derivación¶

    Regla Derivada
    Suma: \(f(x) = g(x) + h(x)\) \(f'(x) = g'(x) + h'(x)\)
    Resta: \(f(x) = g(x) - h(x)\) \(f'(x) = g'(x) - h'(x)\)
    Producto: \(f(x) = g(x)h(x)\) \(f'(x) = g'(x)h(x) + g(x)h'(x)\)
    Cociente: \(f(x) = \frac{g(x)}{h(x)}\) \(f'(x) = \frac{g'(x)h(x) - g(x)h'(x)}{[h(x)]^2}\)
    Cadena: \(f(x) = g(h(x))\) \(f'(x) = g'(h(x)) \cdot h'(x)\)
    Potencia: \(f(x) = [g(x)]^n\) \(f'(x) = n[g(x)]^{n-1} \cdot g'(x)\)
    Exponencial: \(f(x) = e^{g(x)}\) \(f'(x) = e^{g(x)} \cdot g'(x)\)
    Logarítmica: \(f(x) = \ln(g(x))\) \(f'(x) = \frac{g'(x)}{g(x)}\)
